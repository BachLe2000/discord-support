As of December 1st, 2023, Discord will no longer be supported on 32-bit Windows operating systems.

To continue receiving updates and support for your Discord client, you must upgrade to a 64-bit OS version of Windows after this date.

If you are unsure if you are running a 32-bit OS version of Windows, you can check by following these steps:

1. Open your Start Menu, and search for “about”. 2. Select  About your PC to open your System Settings window.

3.  While in System Settings, refer to System Type which will list either a 32-bit operating system or the 64-bit operating system. 

To upgrade your Windows from 32-bit OS to 64-bit OS, you can check out the Microsoft FAQ here. 

Frequently Asked Questions

Q: Will the Discord client I currently have installed continue to work after December 1st, 2023?
A: Basic functionality may continue to work, however functionality and stability is not guaranteed as the client will no longer receive updates.

Q: If I cannot upgrade my computer right now, can I use the browser version of Discord?
A: Yes, as long as your browser meets the minimum requirements found here, you can continue to use it.
