Sometimes you just want to gather with your friends and discuss the latest episode of your favorite show, or play games together. You can easily do all that and more in voice channels! Learn more about not only how to create a voice channel for you and your friends, but also so much more about voice channels!

What are Voice Channels?

A voice channel is a channel in a server where users can connect and chat with one another. You can always tell what channel is a voice channel versus a text channel because a voice channel will be indicated by a speaker icon [].

How to create Voice Channels in a server

There are 2 ways to create a voice channel in a server. Let’s go over both!

Option 1

1. First, tap on the plus (+) next to a channel category in the Discord server you want to make the new channel in.

2. Then tap on Voice and enter a name for the new channel.

3. Tap on Create Channel when you’re ready to create the channel.

4. Congratulations! You now have a new voice channel in your server. If you want to move the channel, you can drag and drop the channel to another category. 

Option 2

1. Right-click on the channel list in the Discord server you want to make the new channel in.

2. Then tap on Voice and enter a name for the new channel.

3. Tap on Create Channel when you’re ready to create the channel.

4. Congratulations! You now have a new voice channel in your server. Creating the channel this way will position the voice channel at the very top, so you can drag and drop the voice channel to move it to a category.

Role permissions for adding a Voice Channel

In order to create voice channels in the Discord server, you either need to be the Server Owner or have Moderator permissions set up. Here’s how to set up role permissions so your Moderators can also create channels.

1. Navigate to the server that you want to change settings for then tap on the server name in the top-left corner of the app.

2. Tap on Server Settings and then head to Roles.

3. Now tap on the role that you wish to edit.

4. Finally, tap on the Permissions tab and locate the Manage Channels permission. If enabled, this role will be able to delete and create text and voice channels. A check mark indicates the setting is enabled, an x means it’s disabled for this role.

How to join a Voice Channel in a server

If a voice channel has a lock icon on it, it means you do not have permissions to join the channel.

1. Navigate to the server that you want to join the voice channel in.

2. Tap on the voice channel that you wish to enter

How to set a Voice Channel status

1. Once you’ve joined a voice channel in a server tap on Set a channel status located under the voice channel name.

2. A pop-up will appear where you can type in your status. You can also add an emoji. 

Role permissions for setting a Voice Channel status

1. Navigate to the server that you want to change settings for, then tap on the server name in the top-left corner of the app.

2. Tap on Server Settings and then head to Roles.

3. Next, tap on the Role you want to update permissions for and then tap on Permissions.

4. Scroll down until you see Voice Channel Permissions and then look for a setting that says Voice Channel Status. 

5. If enabled, the setting will have a checkmark. If disabled, the setting will show an x. If enabled, this means anyone with this role can set a Voice Channel status as long as they are in that voice channel.
