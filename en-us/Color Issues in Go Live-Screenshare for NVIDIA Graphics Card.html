On October 20, 2023, we investigated a stream discoloration issue that occurred during screensharing in Discord for Windows users with the latest NVIDIA GPU driver, version 545.84. The outgoing stream can exhibit a loss in color values, creating a wash out image as you see below. This is a possible video encoder regression in NVIDIA’s 545.84 driver.

If you are impacted by this issue, we recommend upgrading to NVIDIA driver version 546.31 or later as that should resolve the discoloration in your Go Live stream.

You can download 546.31 using the following link: https://nvidia.custhelp.com/app/answers/detail/a_id/5507
To check if a newer driver exists for your system, you can search here: https://www.nvidia.com/download/find.aspx

Frequently Asked Questions

Q: How do I check my NVIDIA GPU driver version?

A: On your Windows device, you can run a DirectX Diagnostic or a DX Diag to find your GPU information along with the driver version under the Display tab of your DX Diag. You can check out this article here for how to run a DX Diag.

Q: How do I install a previous driver version? 

A: Follow the steps below:

  • Download a previous version from NVIDIA’s driver archive
  • Right click over driver installation file and select Run as administrator
  • Press Custom in install dialog
  • Select Perform clean install
