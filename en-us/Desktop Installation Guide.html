When using Discord, we recommend using our desktop clients for the best experience. Here's how to install the Discord app on Windows, MacOS, and Linux. 

What this article covers:

  • Windows
  • MacOS
  • Linux

Windows

To download the Desktop app for Windows, first head over to the Download Page in your preferred browser.

1. Tap on the Download for Windows button to download the app.

2. Follow your browser's prompts to save and open the DiscordSetup.exe to start the installation process.

3. Once installed, you can then open the Discord app and register an account or log into an existing account.

MacOS

To download the Desktop app for MacOS, first head over to the Download Page in your preferred browser.

1. Tap on the Download for Mac button to download the app.

 

2. Open the Discord.dmg file. 

3. Drag the Discord logo and drop it over the Applications folder in the pop-up window.

Linux

To download the Desktop app for Linux, first head over to the Download Page in your preferred browser.

1. Tap on the Download for Linux button to download the app.

2. For manual installation on a Debian based distribution you can use the following commands: 

  • cd ~ /Downloads
  • sudo dpkg -i package.deb or sudo apt install ./package.deb

3. If you’re installing with a package manager, please follow the installation prompts.

4. Now that the app is installed, you can register a Discord account or log into an existing one.
