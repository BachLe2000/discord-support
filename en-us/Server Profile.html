What are Server Profiles?

Server Profiles are customizable information cards that showcase the identity and community of both personal and community Discord servers. Whether it's a friend's private server or a larger community space, these profiles display key details like the server's name, description, icon, games, and unique traits. They help potential members understand what your server is about before joining and appear when users explore or discover servers across Discord.

Server admins and owners can customize their server's profile to highlight unique interests, set expectations for new members, and make their community more discoverable to like-minded individuals.

How to Customize a Server Profile

A Server Profile can only be customized on desktop/browser, however, can be viewed on the mobile app.

To customize your server profile, navigate to your Server Settings and select the dropdown arrow [] next to your server's name. You'll be taken directly to the Server Profile tab.

Navigating to your Server Profile

Here are the customization options available:

  • Name - Choose a clear, memorable identity for your server
  • Icon - Upload a distinctive symbol that represents your community
  • Banner - Pick a banner to add color to your server profile
  • Traits - Select tags that describe your server's focus and  promote discovery
  • Description - Write an engaging overview of your community
  • Games - List the games your community plays or discusses
  • Private Profile - Control who can view your server's details

Name

Your server's name is its identity- it's the first thing people see when they discover your community. Make it clear, memorable, and representative of what your server is about.

To update your server's name, look for the Name field at the top of the Server Profile page.

When you're finished, press Save Changes to confirm your updates.

Icon

Your server's icon is a visual representation of your community that appears in Discord's server list and search results. A custom icon helps your server stand out and makes it instantly recognizable.

To update your server's icon, press the Change Server Icon button or select the current icon in the Server Profile tab.

Press Save Changes to apply your updates.

Banner

If your server is a discoverable server and you set a custom banner, that banner will be displayed instead.

A server banner is a large header color that appears at the top of your server profile. 

To update your server's banner in the Server Profile tab, select a color tile in the Banner section. 

Select Save Changes when you're done.

NOTE: The first banner color option in this menu that appears is based on your current Server Icon’s color.

Traits

Traits are descriptive tags that help categorize and identify your server's focus areas. These tags make it easier for potential members to find communities that match their interests.

To add or edit traits in the Server Profile tab, select a trait box and type your desired trait. You can also include an emoji by pressing the emoji icon [] in the trait box.

Remember to select Save Changes to apply your updates.

Description

Your server's description is a detailed explanation of your server’s personality and purpose. 

To add or edit your server's description in the Server Profile tab, locate the Description field and begin typing in the text box below. 

Press Save Changes to apply your updates.

Games

If your server is focused on gaming, you can specify which games your server plays or discusses. To add or edit games in the Server Profile tab, locate the Games section. You can either select games from the available options or search for specific games by typing in the game field.

You can also remove a game from the list by pressing the checkmark [] n the upper-right corner of the game.

Private Profile

The Private Profile option allows you to control the visibility of your server's profile. When enabled, your server's profile information will only be visible to members who have already joined the server.

This option is enabled by default, however, your profile content becomes viewable to users outside your server when you:
  • Enable server discovery
  • Enable server widget
  • Create an announcement channel
You can re-enable Private Profile through settings, except for one key exception: For discoverable servers, Private Profile cannot be enabled. To set your server's profile to Private Profile, you must first disable server discovery using the steps here.

In the Server Profile tab, find and toggle the Private Profile switch to control visibility.
