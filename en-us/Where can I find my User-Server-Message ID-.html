Users, individual messages, and entire Discord servers all have a multi-digit ID number that can be used in various cases. While you can use Discord entirely without knowing any of these ID numbers, if you are reaching out to our Support team they might require an ID to further assist you. Here’s how to get those ID numbers on both desktop and mobile.

How to Enable Developer Mode

In order to be able to copy an ID number, you need to have Developer Mode enabled. Here’s how.

Desktop

1. Tap on the cogwheel [] in the bottom left corner of the desktop app to open up your User Settings.

2. Then head to Advanced. 

3. Now, tap on the main toggle next to Developer Mode to enable. A “checkmark” means it’s enabled, while an “x” means it’s disabled.

Mobile

1. Tap on your Avatar in the bottom-right corner.

2. Next, tap on the cogwheel [] in the top-right corner.

3. Now, scroll down until you see Advanced under App Settings.

4. Finally, tap on the toggle next to Developer Mode. A “checkmark” means it’s enabled, while an “x” means it’s disabled.

Now that you can see ID numbers, let’s go over how to find each type.

How to find a User ID number

You can also copy the user ID from a server, group chat, or direct message with a user. 

Desktop

1. Navigate to where you want to copy the user ID from. 

2. Next, right-click on the user and tap on Copy User ID. 

3. After copying the user ID you can then paste it where needed.

 

Mobile

1. Navigate to where you want to copy the user ID from. 

2. Tap on the user and then tap on the ellipsis (...) in the top-right corner.

3. Next, tap on Copy User ID. 

4. After copying the user ID you can then paste it where needed.

How to find a Server ID number

Desktop

1. Navigate to the server you want to copy the ID from.

2. right-click on the server icon, and then tap on Copy Server ID.

3. Now you can paste this server ID number where needed.

Mobile

1. Navigate to the server you want to copy the ID number from.

2. Tap on the ellipses (...) next to the server name. 

3. Then scroll all the way down until you see Copy Server ID, tap on it to copy the server ID number.

4. Now you can paste this server ID number where needed.

How to find the Message Link

If you want to copy a message link from a group message instead, you just need to navigate to the group message and copy the ID following the same steps below.

You can use the message link to link a channel (that someone has access to), so they can easily jump into the conversation or for any other reason you might need it. 

Desktop 

1. Navigate to the message you want to copy the link for.

2. right-click on the message and then tap on Copy Message Link.

3. Now you can paste this link anywhere to directly link to the copied message.

 

Mobile

1. Navigate to the message you want to copy the link for.

2. Tap and hold on the message until a menu appears.

3. Scroll down until you see Copy Message Link, and tap on it.

4. Now you can paste this link anywhere to directly link to the copied message.

How to find the Channel ID number

If you want to copy the channel ID number from a group message instead, you just need to navigate to the group message and copy the channel ID following the same steps below.

If you have a channel ID copied, you can send a clickable link to someone by typing out <#channelidnumber>. 

Desktop 

1. Navigate to the channel you want to copy the ID for.

2. right-click on the channel, and then tap on Copy Channel ID.

3. Now you can paste the channel ID as needed.

Mobile

1. Navigate to the channel you want to copy the ID for.

2. Tap and hold on the channel ID until a menu appears.

3. Next, scroll down until you see Copy Channel ID.

4. Now you can paste the channel ID where needed.
