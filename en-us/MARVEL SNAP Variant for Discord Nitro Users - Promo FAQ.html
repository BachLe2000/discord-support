What is MARVEL SNAP?

MARVEL SNAP is a fast-paced collectible card game with innovative mechanics that has won multiple Game of the Year Awards. Build your deck of 12 cards. Each Card represents a Marvel Super Hero or Villain, each with a unique power or ability. The goal of the game is to outsmart and outwit your opponent. Learning to play only takes a few minutes, and matches only take 3-minutes.

How do I claim the Marvel Snap Promo in Discord?

As long as you're a Discord Nitro member or Nitro trial user who meets the eligibility requirements described below and here, you'll be able to claim the Marvel Snap variant code in the Gift Inventory from October 18th, 2023 (9am PST) to November 17th, 2023 (11:59pm PST).

Eligibility for Claiming this Promotion in Discord

Only active Discord Nitro members or Nitro Trial users will be able to claim this code. Unfortunately, if you are a Nitro Basic, Nitro Classic, or Server Boosting member, you'll need to become a Nitro Member in order to claim this promotion.

If you are an active Discord Nitro member, the code must be claimed in the Gift Inventory by November 17th, 2023 (11:59pm PST). 

  • Otherwise, the code will not be available to claim after November 17th, 2023 (11:59pm PST). 
  • Once claimed, be sure to redeem the code by November 30th, 2023 (11:59pm PST). 

Step By Step Claim Instructions 

1) Log into Your Discord account and head to the Gift Inventory tab. 

If you’re a Discord Nitro member or Nitro trial user, start by logging into your Discord account. 

You may notice a notification bar at the top of your browser/desktop app, and you can press the "Take me there" button in the notification bar to be redirected to your Gift Inventory. 

You can also directly head to your User Settings, and then select the Gift Inventory tab.

2) Select the Claim button for the MARVEL SNAP Gift

Once you press Claim on the MARVEL SNAP Gift in the Gift Inventory and you'll then see a new pop-up with your 15 character code for this promotion.

If you want to redeem your code later, you can press the Maybe Later button, and then redeem the code before November 30th, 2023 (11:59pm PST).

How do I redeem the variant from Marvel Snap?

Once you have claimed the code in the Gift Inventory, you'll now need to officially redeem this code on your MARVEL SNAP account (or share this code with a friend).

Eligibility for Redeeming the MARVEL SNAP Variant

All MARVEL SNAP players within eligible regions will be able to redeem this variant. However, you will need to be a Nitro member or have an active Nitro trial to be able to redeem this in-game variant. 

Step by Step Redeem Instructions

In order to redeem the variant on your MARVEL SNAP account, proceed to the MARVEL SNAP website.

Once you're on the website, you'll then need to follow these steps below:

1) Fill in your MARVEL SNAP ID within Player ID Verification and select “Confirm” to verify your username.

Make sure to double-check the username you have entered to be sure it is correct before officially redeeming this exclusive variant. Once redeemed, the variant cannot be transferred to another account.

How to find your MARVEL SNAP ID: 

  • Go to Settings 
  • Scroll to the very bottom 
  • Snap: ID [Tap to reveal Snap Id] 

2) Then, enter your redemption code within the Redeem Code box and select “Confirm”. 

3) Once you have redeemed your code, you can check your rewards through in-game collections via MARVEL SNAP. And that's it! Happy gaming!

Note: Please enter the code accurately and be wary of spacing to ensure a successful redemption. There will not be a confirmation in-game that you have received the variant and title. Go to your Cards and Titles sections to check that you’ve successfully redeemed the rewards. 

If you're running into any issues when attempting to redeem this promotion, reach out to the MARVEL SNAP support team by emailing support@marvelsnap.mail.helpshift.com.  

How do I share my Code with a Friend?

If you're unable to claim this code on your own MARVEL SNAP account, you'll be able to share this code with a friend to use instead.

To share the code, you can copy the 15-character code from the Gift Inventory (in your User Settings in the Discord app) and then share this code to a friend in a Direct Message.
