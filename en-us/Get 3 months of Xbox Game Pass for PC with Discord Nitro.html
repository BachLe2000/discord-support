How do I claim the Xbox Game Pass Code for PC in Discord?

As long as you're a Discord Nitro member, you'll be able to claim the Xbox Game Pass code in the Gift Inventory from June 17, 2024 (10:00am PST) to July 30, 2024 (11:59pm PST)!

Eligibility for the Claiming this Promotion in Discord

In order to claim this code, you'll need to be an active Discord Nitro member, (and you can be either a monthly or yearly Nitro member).

  • If you are an active Discord Nitro member, the code must be claimed in the Gift Inventory by  July 30, 2024 (11:59pm PST).
    • Otherwise, the code will not be available to claim after July 30, 2024 (11:59pm PST). 

Unfortunately, if you are a Nitro trial member, Nitro Basic, Nitro Classic, or Server Boosting member, you are ineligible to claim this promotion.

Note: Xbox Game Pass for PC is only available to new Xbox Game Pass subscribers and Nitro members within eligible regions. If you have claimed your Nitro membership from the Xbox Game Pass promotion, you will not be eligible to redeem a Xbox Game Pass for PC Code in your Gift Inventory!

Step By Step Claim Instructions 

1) Log into Your Discord account and head to the Gift Inventory tab. 

If you’re a Discord Nitro member, start by logging into your Discord account. 

You may notice a notification bar at the top of your browser/desktop app, and you can press the "Take me there" button in the notification bar to be redirected to your Gift Inventory. 

You can also directly head to your User Settings, and then select the Gift Inventory tab.

You may notice a notification bar at the top of your client! Once you’ve selected Claim in the notification bar, you’ll see a Gift magically appear in your Gift Inventory.

2) Select the Claim button for the Xbox Game Pass code for PC

Once you press Claim on the 3 months of Xbox Game Pass for PC in the Gift Inventory and you'll then see a new pop-up with your 12 character code for this promotion.

If you want to redeem your code later, you can press the Maybe Later button, and then redeem the code before August 31, 2024.

We’ll also email you with a copy of your code as well - so if you’re eating pizza and need to redeem it later, we got you! Just search for “Your Xbox Game Pass for PC Redemption Code” in your Inbox. 

How do I Redeem this Xbox Game Pass Code to my Microsoft Account? 

Once you have redeemed the code in the Gift Inventory, you'll now need to officially redeem this code on your Microsoft account (or share this Xbox Game Pass for PC code with a friend)! 

Eligibility for Redeeming the Xbox Code on your Microsoft Account

Only new Xbox Game Pass subscribers can claim the 3 months of Xbox Game Pass for PC! If you have had any type of Xbox Game Pass for PC or Xbox Game Pass Ultimate subscription at any point in time, you will not be able to claim this code! 

If you have previously had Xbox Game Pass for PC or are a current member, you cannot redeem this code. In addition, current Xbox Game Pass Ultimate members cannot redeem this code.

However, while you won't be able to claim this Xbox Game Pass for PC promotion on your own Microsoft account, you can copy and paste the 25 character code and share with a friend!

Step By Step Claim Instructions 

Note: Make sure you are redeeming this Xbox Game Pass for PC on the correct Microsoft account! Discord cannot transfer the Game Pass for PC from one Microsoft account to another!

In order to redeem the Xbox Game Pass for PC promotion on your Microsoft account, you'll need to copy that code from Discord and head over to redeem.microsoft.com!

Once you're at redeem.microsoft.com, you'll then need to follow these steps below:

  1. Sign into your Microsoft Account (if you're not already signed in)
  2. Enter the 25-character code from your Gift Inventory (or from your email) and then press Next
  3. Select a payment method or enter your payment info (Credit/Debit Card or PayPal)
  4. Confirm that you are ready to proceed and all the information is correct by pressing Confirm!

How do I share the Xbox Code with a Friend?

If you're unable to claim this code on your own Microsoft account, you'll be able to share this code with a friend (who has not previously subscribed to Xbox Game Pass for PC or Game Pass Ultimate) to use instead.

To share the code, you can either: 

1) Copy the 25-character code from the Gift Inventory in your User Settings in the desktop or browser Discord client and then share this code directly to a friend or,

2) You can head to your email inbox, search for the email with the subject line "Your Xbox Game Pass for PC Redemption Code", and then copy this 25-character code to share with your friend!
