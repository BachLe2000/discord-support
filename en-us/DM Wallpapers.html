What are DM Wallpapers?

Choosing and setting DM wallpapers is currently an experiment available to select Nitro users only. This feature is not available to users with a Nitro Basic membership.

DM wallpapers allow Nitro users to choose and set a background image from our selection of 15+ images in their direct messages or group chats. All members in a DM can see the wallpaper, regardless of whether they have Nitro, but only Nitro members can select wallpaper options.

How to Set a DM Wallpaper

DM wallpapers are a desktop-only feature that requires dark mode for both viewing and customizing. Learn more about themes here. Users will have one of the two options listed below to access this feature.

Option 1

1. To set a DM wallpaper, right-click a DM or group chat and select Set DM Wallpaper. 

Set DM wallpaper option

2. This opens the Preview a DM Wallpaper window with all available wallpaper options. Choose your preferred wallpaper from the selection.

Preview a DM wallpaper window

3. Press Apply to save your choice.

Setting a DM wallpaper using the preview window

To add a blur effect to your DM wallpaper, select Blur in the bottom-left corner of the preview window.

Blur option in the preview window

Option 2

1. To set a DM wallpaper, navigate to the DM or group chat and select the Set DM Wallpaper icon [] in the upper-right corner. This opens the Preview a DM Wallpaper window with all available wallpaper options.

Set DM Wallpaper icon

2. Choose your preferred wallpaper from the selection.

3. Press Apply to save your choice.

Preview a DM wallpaper window

How to Hide a DM Wallpaper

If you're not the wallpaper setter, hiding a wallpaper will only hide it for you—other users in the DM or group chat will still see it.

For DM or Group Chat Members

If you're a DM or group chat member and want to hide a wallpaper that someone else set, select Hide Wallpaper in the system message.

Hide Wallpaper button in system message

After hiding the wallpaper, you can reveal it again by pressing Reapply.

Reapplying a DM wallpaper

For the DM Wallpaper Setter

To remove a wallpaper you’ve set, open the Preview a DM wallpaper window and select the Remove wallpaper option. Press Apply to save your changes. 

When you remove the wallpaper, it will be removed for all users in the DM or group chat.

Removing a DM Wallpaper Using the Preview Window

Frequently Asked Questions

Q: Is it available on both desktop and mobile?

A: No, this feature is only available on desktop for both viewing and making changes.

Q: When I change the wallpaper, can other chat members see it?

A: Yes, wallpaper changes apply to all members in the direct message. However, members can opt out by selecting Hide Wallpaper in the wallpaper system update message that appears in the chat.

Q: How can I hide a DM wallpaper?

A: Whenever the wallpaper changes, you'll receive a system notification in the direct message. Simply press the Hide Wallpaper button on the right side of this message to hide the wallpaper.

Q: Can I prevent others from changing my wallpaper?

A: When someone changes a wallpaper, it affects everyone in the DM or group chat. If you didn’t set the wallpaper and don't want to see the current one, you can hide it by following the steps here. 

The wallpaper will remain hidden for you, until a new wallpaper is set. If you’ve hidden the wallpaper and want to see it again, select Reapply.
