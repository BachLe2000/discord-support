How to add a friend

The maximum number of friends you can have on an account is 1,000. This includes any pending friend requests you have sent or received.

Adding a friend on Discord is as easy as 1-2-3! Let’s go over the various ways to send friend requests and some common errors you might encounter. 

If you're receiving lots of unwanted friend requests, you may want to ignore certain users. Need help managing unwanted friend requests? Learn how to ignore users by checking out our How to Ignore Users on Discord article.

Username

Desktop/Browser

1. Press on the Discord logo in the top-left corner.

2. Next, make sure you have the Friends tab selected and then press the button that says Add Friend.

3. Here you can type in or paste the username of the friend you’d like to add. 

Note: The username needs to be exact, or the request will not go through.

4. After typing or pasting in the username, press on Send Friend Request to send the friend request.

5. You can tell if the friend request went through by tapping on the Pending tab and seeing it there.

Mobile

1. Tap on the chat bubble [] in the top-left corner of the app.

2. Next, tap on Add Friends to add a friend.

3. Then, tap on Add by Username. 

4. In this Friends list menu, you can type or paste in the username of the user that you want to add. 

5. After you’ve typed or pasted in the username, tap on Send Friend Request.

Profile

Desktop/Browser

1. Navigate to a group chat or a server of the user you’d like to add. 

2. Tap on their profile and then press on View Profile.

3. Next, press on Send Friend Request to send them a request.

Mobile

1. Navigate to a group chat or a server of the user you’d like to add. 

2a. Next, if you’re in a group chat, you can press on the user’s profile to add them in the profile view. 

2b. If you’re in a server, tap on one of the text channels in the server. Now under the Member’s list for the server, tap on the profile of the user you’d like to add.

3. Press on Add Friend to add them as a friend.

Server Member’s List

There is not an option to add a friend through this right-click method in the server member’s list on mobile - you can only add them by tapping on their profile.

Desktop/Browser

1. Navigate to the server or group chat of the user you’d like to add.

2. Right-click on their profile or a message they have sent and then tap on Add Friend.

3. There will not be a confirmation pop-up, however, you can check under the Friends tab to see if the friend request is pending.

 

How to Cancel a Pending Friend Request

If the friend request has already been accepted, you won’t be able to remove the pending request any longer.

Desktop/Browser

1. Press on the Discord logo in the top-left corner of the desktop.

2. Next, make sure you have the Friends tab selected and then navigate to Pending at the top.

3. Here you can tap on the X button next to the Friend Request you want to cancel.

Mobile

1. Tap on the You button on the bottom-right corner.

2. Next, press on Your Friends and then Friend Requests.

3. Here you can see any received or sent friend requests. To remove the request for sent, tap on the X next to the request. 

4. You should receive a pop-up confirmation once it’s been removed and you’ll no longer see the request under your pending list.

Common errors when adding a friend

Friend Request Failed

If you go to add a friend and you receive the warning “Friend Request Failed” this is because the user is not accepting friend requests. They will need to add you as a friend instead - as this means they have their Friend Requests setting disabled. 

Double Check that the Username is Correct

If you receive an error message that states “Hm, didn’t work. Double check that the username is correct.” - then it means the username was not typed or pasted correctly. 

The username needs to be exact, or the friend request will not go through - so double check each number, letter, underscore, or period you’ve typed and then try submitting the request again.

If you’re still running into an issue adding the friend after confirming that you’ve typed their username correctly, you’ll need to reach out to them to see if they can send you a friend request instead.

The Discord Support team is not able to manually add a friend for a user.
