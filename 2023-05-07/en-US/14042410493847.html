What Is Jamspace?

Jamspace is a collaborative whiteboard right inside Discord. You can draw, write text, search for and post GIFs (and can draw on top of those, too)! It’s not what you create, it’s the fun of creating it together!

How To Use Jamspace

Starting Jamspace 

Before starting any Discord activity on Desktop, please make sure to turn on Hardware Acceleration in User Settings > Advanced.

To start a Jamspace activity, follow these steps below!

  • Join a voice channel 
    • Once joined, you should see the Activity Launcher (it looks like a rocket ship) in the voice channel controls.
  •  Click the Activity Launcher to open the Activity Shelf
  • Then select "Jamspace"

  • Once you've authorized Jamspace to access your account, you can begin using it with anyone on the server.

Note: Jamspace is an activity that allows everyone in voice to join (unlimited participants) but is optimized for up to 12 users. More users can join the activity, but performance may decline the more users there are in the activity and interacting with Jamspace.

What GIFs Can Be Used In the Activity?

Jamspace has a built-in GIF integration that you can search from inside the game.

To provide an enjoyable experience for the widest range of Discord users, we may use certain content filters so some GIFs may not appear in the search results.

In this early version of Jamspace your “favorite” GIFs from Discord will not populate in Jamspace.

“Saving” in Jamspace

In this early version of Jamspace there is not a way to save or download an image of the board. There is also no way to load in a saved session. 

If all participants leave the activity the Jamspace board will be wiped – next time you launch the activity you will start with a clean slate to create on. Please take a screenshot of your creations before leaving the activity!

Frequently Asked Questions

Q: Why is Discord adding activities into voice channels?

A: We started Discord because we are all about bringing people together around gaming. As Discord has grown, we've seen more and more people use Discord to come together around games. For instance, we’ve seen people use Discord to not only play League but also to hang out and play more casual, social games with friends like Jackbox and Among Us.

Q: Is there an age requirement?

A: There is no age requirement for Jamspace. By using the Jamspace activity you are agreeing to follow Discord Community Guidelines.

Q: How do I start the activity?

A: To start an activity, first join a voice call from one of the voice channels in your server. Once there, you should see the Activity Launcher button (it looks like a rocket ship) in the voice channel controls. Click the Activity Launcher to open the Activity Shelf, and then select "Jamspace.”

Q: What platform is Jamspace available on?

A: Jamspace can be used on the Discord desktop app, web browser, or mobile and tablet apps.

Q: Can I download an image of my Jamspace board?

A: In this early version of Jamspace there is not a way to save or download an image of the board. There is also no way to load in a saved session. 

If all participants leave the activity the Jamspace board will be wiped – next time you launch the activity you will start with a clean slate to create on. Please take a screenshot of your creations before leaving the activity!

Q: Who can edit/delete things I’ve made in Jamspace?

A: Anyone in the activity session can draw, delete objects, and move or resize any object on the board. You can only edit text you have written, you cannot edit text that others have written.

Q: How do I view/delete my Activities on Discord data?

A: To enhance the activity experience, Discord Activities access and temporarily store the following user data once a user joins the Activity:

  • Discord ID
  • Avatar
  • Username

For Activities played in a single session, user data is stored so long as the Activity remains active, even if the user leaves the Activity. Once all users leave the Activity and it is determined the Activity has ended, the user data is deleted and no longer retained.

For Activities that can be played across multiple sessions (like with Chess in the Park or Checkers in the Park), the user data above is retained. 

Deleting your data will remove all of your progression across these Discord-developed activities, however Jamspace does not have any game progression or level data so this does not affect Jamspace like it would other activities that do have level progression.
