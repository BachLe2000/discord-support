Custom Profiles

We’ve added a User Profile page in User Settings where you can customize your profile for how it appears to other Discord members. 

Looking to change your username or custom tag? Visit our account customization page. 

What does this article cover?

  • Profile Avatar
  • Profile “About Me”
  • Profile Banner
  • Profile Theme
  • Custom Profile on Mobile

Profile Avatar

Style your profile by uploading an avatar! Supported file types are PNG and JPG.

Note: Nitro subscribers can upload animated GIFs as their profile avatar.

How to Change Your Avatar

Step 1: Go into your User Settings by clicking the Cog/Gear Icon (⚙︎) at the bottom-left part of the app

Step 2: Navigate to the User Profile tab

Step 3: Press either Change Avatar button

Note: Looking to personalize your account identity in each of your servers? You must be a Nitro subscriber to unlock Server Profiles.

Profile “About Me”

You can share a little about yourself in your profile card in the About Me section. 

Your About Me can have:

  • Up to 190 characters
  • Emojis
  • Markdown text and links
  • Unicode character

Note: Nitro and Nitro Classic subscribers can use custom emojis in their About Me.

1. Go into your User Settings by clicking the Cog/Gear Icon (⚙︎) at the bottom-left part of the app

2. Navigate to the Profiles or press the Edit User Profile button on the My Account page.

3. In the User Profile tab, fill out the text field under About Me. Don’t forget to Save!

Profile Banner

Nitro members can upload an image or animated gif as their profile banner. Supported file types are PNG, JPG, and animated GIFs under 10MB. The recommended minimum size is 600x240.

Tip: Images can be cropped after upload (GIFs cannot be cropped)

Profile Theme

Nitro members can set a profile theme that will stylize their mini-profile card that anyone can see when they press on their avatar icon. As a default, the profile theme will reflect the client theme selected in the Appearance tab.

You can select custom profile theme colors by pressing the Primary and Accent color selection boxes and choosing or entering HEX color codes.

1. Go into your User Settings by clicking the Cog/Gear Icon (⚙︎) at the bottom-left corner of the app

2. Navigate to the Profiles or press the Edit User Profile button on the My Account page.

3. In the User Profile tab, go to the Profile Theme section, press on the Primary color square to choose the main profile color.

4. Press the Accent color square to choose a second accent color for your profile theme. 

5. Don’t forget to press Save!

Custom Profile on Mobile

1. Go to your User Settings by swiping right then tapping on your profile avatar icon in the bottom right corner.

2. Select User Profile.

3. Customize features you have access to, like your Profile Avatar, About Me section, Profile Theme, or Custom Profile Banner.
