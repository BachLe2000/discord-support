Managing server roles and permissions can be complex. It's not always clear how permission changes will affect different members' experience of your server.

The View as Role feature helps you understand exactly how your server appears to members with specific roles. This powerful tool lets you preview your server from different role perspectives before making changes.

What this article covers:
  • Who Can Use View Server As Role
  • What You Can Preview with View Server As Role
  • How to Use View Server As Role
    • Viewing Multiple Roles

Who Can Use View Server As Role

The View Server As Role feature is available to server owners and members with Administrator permissions. If you don't have Administrator permissions, you won't see this option in your Server Settings.

Administrator is the most powerful role permission. It grants full access to all permissions, so be sure to assign this role with extreme caution.

What You Can Preview with View Server As Role

View Server As Role lets you check exactly what different roles can access, including:

  • Visible text channels and categories
  • Accessible voice channels
  • Available server settings and management options per roleÂ 
NOTE: Because voice restrictions are tied to the user, View Server As Role doesn't reflect voice restrictions like suppression or priority speaker.

How to Use View Server As Role

1. To access View Server As Role, press the server name in the upper-left corner and select Server Settings.

2. Navigate to the Roles tab to see all server roles.

How to Navigate to Roles in Server Settings

3. Select your desired role and scroll to View Server As Role in the Display tab.

View Server As Role Permission in Display Tab

You can also select the ellipsis [] next to any role name to enable View Server As Role.

Using the Ellipsis Menu to View Server As Role

Viewing Multiple Roles

When View Server As Role is active, a control bar appears at the top of Discord. This lets you preview how the server appears to members with multiple roles.

View Server As Role Banner

Here, you can also press Select Roles to see all available roles in your server.

Select multiple roles to see the combined permissions view. You can also use the search bar to quickly find specific roles.

Selecting Roles within View Server As Role Banner

Exiting View as Role

To return to your normal view, press the Disable button in the View as Role control bar. Your server view will return to normal.

Disable Button in View Server As Role Banner

Your server view will return to normal. To quickly return to role settings, press the Back to Roles Settings button in the top-left corner.

Back to Roles Settings Button in View Server As Role Banner
