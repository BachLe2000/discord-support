Finding Your Server Notification Settings

1. Go to the server where you want to adjust notifications and press the server name in the upper-left corner to open the dropdown menu. You can also right-click the server name to open this menu.

2. Select Notification Settings from the dropdown menu.

3. The Notification Settings menu will appear, where you can customize your notifications for this server.

Accessing Server Notification Settings on Desktop

To find your server Notification Settings on mobile, tap a server’s name, then tap the Notifications icon [] > Server Notification Settings.

How to Mute an Entire Discord Server

In your server Notifications Settings, you can mute an entire Discord server by simply toggle the Mute Server to silence all channels in that server at once. You can also choose how long to mute the server notifications by selecting an option from the Mute duration dropdown menu.

To learn more about managing unwanted notifications, including how to ignore users, check out our Help Center article on How to Ignore Users on Discord.

Muting an Entire Server in Notifications Settings on Desktop

Muting a server will affect:

  • All notifications across the server
  • All channels within the server
  • All server activity and updates

However, you will still receive @everyone mentions.
Please note that mobile push notifications will also be disabled when the server is muted.

To mute an entire server on mobile, tap a server’s name, then tap the Notifications icon [] and toggle the mute option. You can also select how long you’d like to mute the server from the Mute this Server list.

Server Notification Settings Options

Aside from muting an entire server, you can instead choose different levels of notifications. Simply select an option in the Server Notifications Settings list.

On mobile, navigate to Notifications and tap the option you’d like in Server Notification Settings.

Selecting an Option from the Server Notification Settings List

Here, you can also choose suppression options and more:

  • Suppress @everyone and @here: Prevents notifications from @everyone and @here mentions across the entire server.
  • Suppress All Role @mentions: Prevents notifications from all role mentions across the entire server.
  • Suppress Highlights: Controls email notifications for the best content from servers you follow. This toggle works independently of other notification settings - you can receive Highlights even with notifications off, unless you mute the server completely. Learn more.
  • Mute New Events: Prevents notifications for new events across the entire server.
  • Mobile Push Notifications: When you're signed in on your Android or iOS device, you'll get push notifications for anything you'd normally be notified about on desktop. Pretty convenient!

Finding Your Channel Notification Settings

Discord's channel notification settings provide granular control over alerts at the individual channel level. You can override notifications for specific channels through Notification Overrides, even when other server notification settings are enabled.

1. Simply navigate to your server Notification Settings > Notification Overrides.

2. Select the dropdown menu to see a list of channels and categories. You can customize notification settings for each channel, even if you have other server-wide settings enabled.

3. Select the channel you want to customize and choose your notification preferences:

  • All: Receive notifications for every message
  • Mentions: Get notified only when you're mentioned
  • Nothing: Disable all notifications
  • Mute: Silence everything including @mentions

This gives you precise control over each channel's notifications, regardless of your server-wide settings.

Customizing Channel and Category Notifications Using Overrides

You can also set your channel notification settings by right-clicking a channel > Notification Settings.
