What are per-server profiles?

Per-server profiles are a new customization feature for Nitro users! With per-server profiles, users can set a different server-specific avatar, banner, and about me section for each of the servers they're in.

How to Customize Your Profile for Different Servers

NOTE: Even if you change your per-server profile, your user profile will be unaffected and remain as your primary avatar in servers where you don't upload server avatars.

How can I change my Server Avatar?

Desktop/Browser: Right-Click on your Avatar

1. In the browser or desktop app, you can right-click your avatar in a text channel. This will activate the drop-down menu.

2. Select Edit Per-server Profile to navigate to the Per-server Profiles page. 

3. You can then press on your avatar or Change Avatar to upload the specific avatar you would like to use for that per-server profile.

Like profile avatars, PNG and JPG are the supported file types and Nitro users can upload a GIF for their server avatars.

Right-click your avatar and select Edit Per-server Profile to customize it for specific servers

4. Once you have chosen your avatar, press Save Changes to confirm. 

Save Changes Button

After uploading a server avatar, you can crop, center, and rotate PNG, JPG, and GIF files.

Edit your avatar: Crop, center, and rotate your profile picture

Desktop/Browser: Left-Click on your Avatar

1. Alternatively, you can left-click your avatar in the server’s Member List, which is the sidebar located to the right, to open the mini profile pop-up. You can also open this when left-clicking on your avatar within a text channel, or in a voice channel.   

2. You'll then see an Edit Profile option that you can press.

3. Select Edit per-server profile to change your server avatar in the per-server profile page.

Left-click your avatar in the Member List and select Edit Per-server Profile

4. When you’re all set, press Save Changes to confirm your avatar change.

Save Changes Button

Desktop/Browser: Resetting My Server Avatar to a Recent Avatar

Recent Avatars lets Nitro subscribers store their 6 most recent avatars, while non-subscribers can store 2.

To return to a recent avatar, navigate to the Profiles page. You can do this using either of the methods described above. 

If you'd like to use a different recent avatar instead, select Change Avatar and choose one from the Recent Avatars section.

After making your selection, press Apply and Save Changes to confirm.

Save Changes Button

Mobile: Changing your Server Avatar

You can customize your server avatar on iOS or Android through two methods: User Profile and Per-server Profiles.

Through the User Profile: 

1. Navigate to the specific server and select your avatar from either a channel where you’ve posted messages or from the server's Member List.

2. Tap Edit Per-server Profile to access your Per-server Profiles settings.   

3. In the Per-server Profiles page, tap your current server avatar and select Upload Image to set a new avatar.

Per-server Profiles on Mobile

4. After a new server avatar has been uploaded, tap Save in the top right corner to confirm the change.  

Remember to tap Save on mobile devices to confirm your profile changes

Through Per-server Profiles: 

1. Tap on the server’s name and select Edit Per-server Profile to open your Per-server Profiles settings. 

2. Select your current server avatar and tap Upload Image to choose a new image.

Uploading an image in per-server profiles

3. After making your changes, tap Save in the top-right corner to confirm.

Tap Save on mobile devices to confirm your profile changes

How do I change my profile theme?

Nitro members can customize each of their per-server profile themes! This will specifically change the colors of your mini profile card. You can choose a Primary color and an Accent color.

On Desktop/Browser

In your Profiles tab > Per-server Profiles page, under the Profile Theme, press the Primary color square to choose the main color of your mini profile card. Then, press the Accent color square to choose a secondary color.

Select Profile Theme Colors: Choose a primary and accent color on desktop

On Mobile

To access your Per-server Profile page on your mobile clients, swipe right while in the Discord app and tap on your avatar icon in the bottom right corner to open up your profile. Then, tap Edit Profile.

To customize your User Profile and per-server profiles, tap your avatar and Edit Profile

You can also tap on your avatar icon in a channel of your desired server and select Edit Per-server Profile in your mini profile card.

In the Profile page, make sure the profile option slider is towards the right for Per-server Profile and that server you intend to customize your per-server profile is chosen in the dropdown menu.

Per-server Profiles Tab on Mobile

Under the Profile Themes section, press the Primary color square to choose the main color of your mini profile card. Then, press the Accent color square to choose a secondary color.

Pick a Primary and Accent color under Profile Theme

Please note: if you have Sync profile themes enabled in your Accessibility settings page, which syncs your profile themes to your chosen Discord client theme, the background color of your profile card will reflect the color of your client theme (Light or Dark).

How do I change my profile banner?

Head over to your User Settings > Profiles > Per-server Profiles and press Change Banner to upload your server banner.

Changing your Profile Banner on Desktop

Nitro members can upload an image or animated GIF as their profile banner. Supported file types are PNG, JPG, and animated GIFs under 10MB. The recommended minimum size is 600x240.

How do I set up an “About Me” for my Per-Server Profile?

In the Per-server Profiles settings page, you can write a small blurb in the text box under About Me that’s specific to your per-server profile.

Your About Me for your per-server profile can have:

  • Up to 190 characters
  • Markdown text and links
  • Emojis
  • Unicode
NOTE: Nitro and Nitro Classic members can use custom emojis in their About Me.

About Me section in Profiles

How do I add avatar decorations and profile effects to my per-server profile? 

1. In your Per-server Profiles settings, be sure to choose the server you would like to update within the Choose A Server selection box. 

Choose Server dropdown in Per-server Profiles

2. Once you have chosen the server, press Change Decoration or Change Effect to select the avatar decoration or profile effect you would like to set for the per-server profile. 

Example of Adding an Avatar Decoration and a Profile Effect

3. When you’re ready, be sure to confirm your changes by pressing Save Changes in the pop up box located at the bottom.

NOTE: Avatar decorations and profile effects can be found in the Shop. You must purchase a profile cosmetic before being able to add to your profile.

Frequently Asked Questions

Q: What happened to server nicknames?

A: Server nicknames still exist and are free to all Discord users! To change your server nickname, you can follow the same steps to change your server avatar, but instead input/change your server nickname in the in-app pop-up window.

Learn more about server nicknames in this article!

Q: Do I get custom server avatars with Nitro Classic?

A: Nitro Classic users can still set server nicknames; however only Nitro users will be able to set custom server avatars.

Learn more about Nitro and Nitro Classic in this article!

Q: What are the size limits and accepted file types for server avatars?

A: Accepted file types for server avatars are JPG, PNG, and GIF files. The maximum file size is 8 MB.

Q: Can I edit avatars after I upload them?

A: You can crop, center, and rotate files after uploading them.

Q: Is there a limit to the number of custom server avatars I can have?

A: No, but there is a limit to the number of servers you can join. Nitro users can join up to 200 servers.

Q: Can I have more than 1 avatar in a server?

A: At this time, you may only set 1 custom avatar per server.

Q: Can I reset all my server avatars at once?

Unfortunately, there is no bulk editing option at this time. You will need to manually reset your avatar in each server.

Q: Can I change my server-specific avatar decoration and profile effect on my mobile device?

A: At this time, you can only change server-specific avatar decoration and profile effects on the desktop/browser app.

Q: Will I get in trouble for using an inappropriate image as a server avatar?

As is true for all content that you use on Discord, you are responsible for ensuring that any content you use for server avatars complies with our Terms of Service and Community Guidelines.

Q: Are server owners or admins able to change another server member's avatar like how they can change nicknames?

No, server owners and/or admins do not have the ability to change a server member's server avatar or reset it to their primary avatar.

If you notice someone using a server avatar that violates our Terms of Service or Community Guidelines, please use our report form to report the user to our Trust and Safety team.
