Whether this is the first or tenth server you've made, you’re in the right place for going over the server creation process! Before we begin, keep in mind that the Add a Server () icon can be hidden if you have a lot of servers. Simply scroll down the server list on the left side of the app to find where the Add a Server () icon is hiding.

After you select the Add a Server button, you will see a pop-up window that will give you the option of choosing a blank slate for your new server or a template. It also has a spot at the bottom of the pop-up if you have an invite code you’re trying to use. For this example, we’ll be creating a server using the Create My Own option.

Not sure how you want to customize your server? You can always download a Server Template! Check out this Help Center article for more information.

We try to make it easy for you by offering to guide you through a set up for a community or friends server, but you can always skip if you’re unsure - no need to commit now!

Next, you can enter in the server name, and change the server icon by locating it in your devices files. Everyone knows the name and icon are the most important details for a server!


The minimum size for icons is 128x128. If you'd like to select a specific area from a large image, crop it in multiples of 128 (128X2=256, 128X3=384, etc.).

Press "Create" and you’ve started a brand new server. Now, send your friends an invitation and start chatting!

Don’t feel pressured to perfect your server in the creation process - you can always update the name and icon of your server at a later time!
