Understanding Server Member Applications

The server member applications feature is rolling out gradually and can only be enabled through the desktop app. However, users applying for a server can submit or withdraw their applications using either desktop or mobile devices.

People can join your server in three ways:

  • Invite only
  • Through Discovery
  • Member application

Using member applications, server owners can precisely control who joins their server. Simply navigate to your Access settings, by pressing the server name in the upper-left corner. Then select Server Settings > Access.

How to Configure Server Join Settings

In Access settings, you can control your server's privacy level by choosing how people join. Select either Invite Only, Apply to Join, or Discoverable under the How can people join your server? section.

Server Join Setting Options: Invite Only, Apply to Join, and Discoverable

Invite Only

Invite Only is the default privacy setting for all Discord servers. This gives you maximum control over who joins your server:

  • Members can only join through valid invite links
  • Enable server rules to require new members to accept your server guidelines before participating
  • Generate customized invite links with expiration dates and usage limits

Invite Only Option

Apply to Join

While applications are pending, users cannot view any server content.

With the Apply to Join option, you can screen potential members by requiring them to complete an application form before joining your server. Here's what you need to know:

  • All applications must be reviewed and approved by server administrators before members can join
  • Create custom questions to ensure new members are a good fit for your community
  • Choose whether to display application links on server profiles for easier discovery

Apply to Join Option

NOTE: Server members who have the Kick Members permission enabled can create special invite links that bypass the application process. These invite links can be viewed in Server Settings > Invites.

Discoverable

The Discoverable option lets you enable Server Discovery, making your server visible to all Discord users who are searching for new communities:

  • Your server must be set up as a Community server to be discoverable
  • You'll need to meet Discord's discovery eligibility requirements including safety standards
  • Anyone can join and find your server directly through Server Discovery

Discoverable Option

Reviewing Server Member Applications

If your server’s privacy level was set to Apply to Join, server administrators with the appropriate permissions can manage member applications through the Members page. You can learn more about the Members page here.

If a user has applied to join your server, you will notice a notification located next to the Members tab located above the server’s channel list. In the Members page, select the user’s server member application.

You will be able to review the following information:

  • Why they would like to join your server
  • When they joined Discord
  • When they submitted their server member application

Reviewing a Server Members Application in Members Page

Processing Server Member Applications

When reviewing server member applications, you have three processing options:

  • Approve: This grants the user access to your server.
  • Reject: This denies the user access to your server. To permanently prevent a user from reapplying, you can ban them from the server.
  • Interview: You can choose to directly message and interview the user before making your final decision to approve or reject their application.

Frequently Asked Questions

Q: How long does it take to review server member applications?

A: The review time varies depending on your server administrators' availability. There is no set timeframe.

Q: Can users edit their server member applications after submitting?

A: No, users cannot edit their applications after submission. However, they can withdraw their application and submit a new one if needed.

Q: Who can review server member applications?

A: Server administrators and members with appropriate permissions (specifically those with the Kick Members permission) can review and process member applications.

Q: What happens if a server member application is rejected?

A: When an application is rejected, the user will be notified and will not gain access to the server. They can choose to apply again unless they have been banned by server administrators.

Q: Can I customize the server member application questions?

A: Yes, server administrators can create custom questions to better screen potential members and ensure they're a good fit for the community.

Q: Do server member applications work on mobile devices?

A: While the feature can only be enabled through the desktop app, users can submit and withdraw applications using either desktop or mobile devices.
