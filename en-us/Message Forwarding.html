What is message forwarding?

This is currently an experiment on iOS, Android, and desktop and is not available to everyone.

Have you ever wanted to send your friends that cool thing you saw on another server or send them a funny meme from another channel?

Forwarding will let you to select any message or piece of media and send it to another place in Discord. 

A forwarded message will show up clearly labeled as "forwarded" with only the content of the message included.

How to Forward a Message

Desktop/Browser

1. To get started, hover over to a message and press the Forward button [] within the hover bar in the upper right corner of the message.  

Others may see the forward button next to media and embeds. Using this option will only forward the media or embed, not the entire message. To forward the entire message, navigate to the desktop/browser app and either:
- Open the context menu by right-clicking the message and selecting Forward.
- Hover over the message and select the Forward button in the hover bar.

2. The Forward To pop-up menu will appear, and you can select where you would like to forward the message. 

Note: You can forward the message to a maximum of 5 places.

3. Once you’re all set, press Send in the lower right corner of the pop-up menu to forward your message. 

Your forwarded messages will be labeled as “Forwarded” without any indication of the message’s original author. The original message's source destination will only show as a link to those who already have access to that server or if the server is public. 

Mobile

1. To get started forwarding a message on mobile, long-press the message and select the Forward option in the context menu. This action will forward the entire message. 

Others may see the Forward button [] located next to media and embeds. Using this option will only forward the specific media or embed, not the entire message.

. 

2. The Forward To menu will appear, allowing you to select where you would like to forward your message. Once you’re ready, tap Send. 

3. Your message will be forwarded to the selected places and be labeled as Forwarded. 

Frequently Asked Questions 

Q: Why can’t I see the option to forward a message?

A: Message forwarding is currently an experiment and is available to a limited number of users.

Q: Why can’t I forward certain messages? 

A: Messages from monetized channels cannot be forwarded as well as messages from  age-restricted channels to unrestricted channels. 

Q: Can I publish a forwarded message in an announcement channel? 

A: No, publishing a forwarded message in announcement channels is not allowed at this time. 

Q: Will others see who originally wrote the message I forward? 

A: No, the original author of the message will not be shown: only the content is included. .

Q: Can everyone see where a forwarded message came from? 

A: No, the link back to the original message is only visible to those who have access to the original server and channel, or if it’s from a public server. 

Q: If the  original message is edited later, will my forwarded message update too? 

A: No, the forwarded message is a copy of the original message at the time you forwarded it and will not update. 

Q: Can I add my own text along with the forwarded message? 

A: No, this option is not available at this time. 

Q: Can I disable forwarding for my own messages or for messages in/out of my server? 

A: No, you cannot control this at this time.
