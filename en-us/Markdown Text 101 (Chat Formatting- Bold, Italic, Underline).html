Want to inject some flavor into your everyday text chat? You're in luck! Discord uses Markdown, a simple plain text formatting system that'll help you make your sentences stand out. Here's how to do it! Just add a few characters before & after your desired text to change your text! Let’s go over some examples.

What this article covers:

Text Formatting

While Discord doesn't have built-in font options, you can customize your message formatting using these markdown options below!

You can also use unicode characters to add some fun and flare to your text!

Italics *italics* or _italics_ Underline italics __*underline italics*__
Bold **bold** Underline bold __**underline bold**__
Bold Italics ***bold italics*** underline bold italics __***underline bold italics***__
Underline __underline__ Strikethrough  ~~Strikethrough~~

Organizational Text Formatting

Headers

Don’t forget to add a space between the leading heading character (#, ##, ###) and your text!

To create a header you just need to include a specific number of the hash/pound sign character (#). Use (#) for a big header, (##) for a smaller header, or (###) for an even smaller header as the first character(s) in a new line to make a header. Here is an example of what each header type looks like.

For example: 

# Big Header

## Smaller Header

### Even Smaller Header

Example of Using Message Headers 

Subtext

Like headers, you can add subtext to any chat message. To do so, add a (-# ) before the text you want to appear in the subtext. Don’t forget the space after # before your message.

For example: 

-# All members of that team must be present and in the Discord voice channel by 6:55PM PST! Be sure to go live and turn on your webcams by 7PM PST for us to begin!

Example of Adding Subtext

Subtext works very similarly to Headers, so the (-#) must be at the very beginning of the line in order for it to work.

Masked Links

This is only available for bots and webhooks.

You can use masked links to make text a clickable or pressable hyperlink. To do so, you need to include the text you want displayed in brackets and then the URL in parentheses with no space between the brackets and parentheses.

For example: 

[Link Text](http://www.discord.com).

Example of Masking a Link

Lists

Don’t forget to add a space between the list bullet (-, *, 1., etc) and your text!

You can create a bulleted list using either (-) or (*) in the beginning of each line.

For example: 

- List of Stuff

- and more things

- and even more things

Example of How to Create a Bulleted List

You can also indent your list by adding 2 spaces before (-) or (*) at the beginning of each line.

For example:

- This is another list

- of stuff

- and more things

Example of How to Indent a Bulleted List

Code Blocks

You can make your own code blocks by wrapping your text in backticks (`).

For example: 

`Wow look at this one live code block` 

Example of a Code Block

If you want to create a multi-line code block, you can do so by wrapping your text in (```), like this beautifully written haiku.

For example:
```haikus written here
really shouldn’t cause conflict
markdown is more fun```

Example of a Multi-line Code Block

Block Quotes

Don’t forget to add a space between the leading > and your text!

Another option is using block quotes. To use a block quote, you just need to put (>) at the beginning of a line of text to create a single block quote. 

For example:

> Let’s get that bread

Example of Creating a Single Block Quote

If you want to add multiple lines to a single block quote, just add (>>>) before the first line.
For example:

>>> pink lemonade

orange juice

tea and water

Example of a Block Quote with Multiple Lines

Spoiler formatting is another useful feature! Using the syntax || around your text will mark it as a spoiler. Note that this doesn’t work within code blocks. You can learn more about Spoiler Tags here!

How to Remove Formatting

It is currently not possible to disable markdowns, however, you can add a backslash (\) before the markdown to remove formatting.

For example, if you’re wanting to create a list without the formatting you should add a backlash (\) before each line.

Backslashing before a markdown will remove the markdown formatting
