Server Guide Beta

This feature is currently in beta for a certain percentage of servers. You can check to see if you have it available by going to your Server Settings > Onboarding page.

Joining a new server is an exciting experience, but sometimes it can be overwhelming to familiarize yourself with all the different channels and rules. To help new members with their onboarding experience, admins can now add a Server Guide.

What this article covers:

  • What is a Server Guide?
  • How to set up Server Guide
  • Frequently Asked Questions

What is a Server Guide?

Server Guide is a dedicated space that new members see after answering the pre-join questions in a server’s Community Onboarding experience. Its purpose is to enable server admins to customize their welcoming experience further by offering an engaging and informative introduction to their community.

By using Server Guide, admins can showcase their server's unique features, rules, and channels, ensuring that newcomers feel welcomed and informed from the moment they join. This tailored experience helps new users quickly acclimate to the server.

How to set up Server Guide

Server Guide is a feature within Onboarding, which can be accessed via Server Settings > Onboarding.

Once you’ve completed the Default Channels and Questions portion of Onboarding, you can advance to the Server Guide setup. Here, you can customize a few things:

  • Welcome Sign: a space at the top of Server Guide where you can write a custom message from you, the admin.
  • New Member To Do’s: 3-5 tasks for your new members that help them get started talking and engaging in your channels.
  • Resources: Turn read-only channels into fancy resource pages in your Server Guide. These will no longer appear on the channel list and come with some perks:
    • Users are directed to the top of pages instead of the bottom of a message thread
    • Message bars and avatars are removed so it looks cleaner
    • All the content, embeds, media, and formatting will stay the same

Frequently Asked Questions

Q: Which servers have access to Server Guide?

A: Server Guide is currently in a limited beta that includes 50% of community servers.

Q: I don’t have access to Server Guide. When will I get it?

A: We are using this testing period to evaluate performance and feedback from communities. We’ll share any updates around availability via Community Updates. 

Q: Why is Server Guide only available through Onboarding?

A: Currently, Server Guide has been built as a component within Onboarding. We see Server Guide as one part of creating a more informative, inviting Onboarding experience for new members.

Q: Will all of my new members see Server Guide once I enable it?

A: During this beta, 80% of new members that join a server with Server Guide enabled will see it, while 20% will continue seeing the previous experience.

Q: How does Server Guide work with the Welcome Screen?

A: When a server enables Server Guide, new members won’t see a Welcome Screen.

Q: How does Server Guide work with Rules Screening?

A: Rules Screening still appears once a new member leaves Server Guide to visit a channel.
