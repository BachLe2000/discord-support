What is the policy for Server Ownership Transfers?

Our Discord support team can only assist with server ownership transfers if the following criteria are met:

  • The server has 100 members.
  • The current server owner hasn't accessed their account for at least a month.
  • You (the requestor) have been active on your account in the last 30 days.
  • You (the requestor) have administrator or moderator permissions within the server.
Our Support team can only assist with server ownership transfers if your request meets the criteria. If your request does not meet the criteria, then it will be denied— no exceptions.

How do I submit a Server Ownership Transfer Request ticket?

1. Visit us at https://support.discord.com/hc/requests/new

2. Under What Can We Help You With?, select Help & Support

3. Input the email associated with your Discord account

4. Under Type of Question?, select Server Related Requests

5. Under Server Related Requests, select Server Ownership Transfer Request

6. Input the requested information

What information is needed in a Server Ownership Transfer request?

In order for our support team to review your request, we will need you to submit the following information. Without the information below, we will be unable to proceed further with reviewing this request. If invalid information is provided, your request may be delayed and follow-up will be required.

  • Server Name
  • Server ID
  • Inactive owner's Discord Username
  • Inactive owner's User ID
  • New owner's Discord Username
  • New owner's User ID

How do I find a Server/User ID?

1. You must enable Developer Mode on your Discord app before you can find the server/user ID. You can check out this support article on how you can enable developer mode on your Discord app.

2. After you’ve enabled Developer Mode, you can now find and copy a server ID and a User ID.

To find the user ID, right-click a user’s avatar and then select Copy User ID. You can also press on a user’s avatar > press on the ellipses icon in the top right corner of the user’s profile > select Copy User ID.

To find a server ID, right-click the server’s name or the server icon and then select Copy User ID.

Please note that we cannot transfer ownership if you’re asking on behalf of another user, and you will need to ask that other user to directly contact our support team with their own transfer request! The new owner should be the one submitting the support ticket, unless your server is Partnered or Verified.

What if my Server is Partnered or Verified?

Before submitting a ticket, please check if your server is part of either the Partnered or Verified programs. If your server is Partnered or Verified, you will have the corresponding badge next to your server’s name. You can learn more about server types on this support article.

Please note that if your server has the Community Feature enabled, then this is not included in the Partner or Verified program. When submitting your transfer request, please only select Yes under Is your server Partnered, Verified? if your server is confirmed to be Partnered or Verified, not Community. Choosing incorrectly could delay your request processing.

After confirming that the server is part of our Partner or Verified program, only the server owner can submit an ownership transfer request.

What if I Want to Request an Ownership Transfer of Multiple Servers?

Our team can only handle one server per request. Therefore, if you want to request ownership of multiple servers, we ask that you file a separate ticket for each one.

Will the Inactive Server Owner be Notified of the Server Transfer?

If a server owner is inactive and not banned, then we do contact them to get their consent for the server ownership transfer. The current server owner has 7 days to respond to our inquiry. If they approve the transfer or they don’t respond within 7 days, we transfer the server if eligible per our requirements. If the current owner denies the transfer request or the criteria for transfer is not met, we will not be able to proceed with the transfer of the server.

A Note on Data and Your Privacy

In line with data minimization principles, information provided by you in your server ownership transfer request will be deleted after the transfer is complete.

We take your privacy and data security seriously. The information collected during the server ownership transfer process, including the server name, server ID, inactive owner's Discord username and user ID, and the new owner's Discord username and user ID, will be used solely for processing your transfer request. This data will be handled confidentially and in accordance with Discord’s Privacy Policy and applicable data protection laws.
